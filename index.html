About Butns

Butns is an advanced buttons library for your website, using a set of pre defined classes you can fully customize your button and step up your website's design.

Butns Reference

Inspect any button and check the classes to see how it's done!

Installing Butns

You have many options to choose from when it comes to installing Butns

  • CDN: <link rel="stylesheet" href="https://unpkg.com/butns">
  • npm: npm install butns
  • bower: bower install butns
  • clone the repo: git clone git://github.com/ahmedtarek2134/Butns
  • download the zip file from the github repo

Available colors in Butns

Butns features all of Google's Material Design Colors with two shades of every color, a dark and a light one.

Available button types in Butns

There are many button types to choose from in Butns making it more versatile

Note: If you want the fill style, don't specify a type class since it is the default one.

Note: for a dark or a light outline, you need to specify two type classes, outline and dark or light

Available options in Butns

There are many options to add to your button to give it a different look and feel.

Using Butns

Here's how to get in with using Butns

  1. Link the butns stylesheet in your html: <link rel="stylesheet" href="path/to/butns.css">
  2. Start by adding the class of "butn" to a button or an a tag: <button type="button" class="butn">Butn</button>
  3. Choose your color and add it as a class: <button type="button" class="butn deep-purple">Butn</button>
  4. Choose your type and add it as another class: <button type="button" class="butn deep-purple gradient">Butn</button>
  5. Choose your options and add them as another class: <button type="button" class="butn deep-purple gradient semi-transparent">Butn</button>

As you can see, butns is all about specifying how your button will look like using class names, its super easy to use!

Note: all class names should be in lower case

Example of using two types, dark and outline: <button type="button" class="butn deep-purple dark outline">Butn</button>

Example of using more than one option: <button type="button" class="butn deep-purple rounded large shadow">Butn</button>
